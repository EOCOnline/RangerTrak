To import a pipe-delimited text file to Excel, open the file from within Excel. The Excel Text Import Wizard will guide you through the process of converting the text file to a spreadsheet format. Choose "Delimited" as the format. In the next screen, choose Other as the delimiter, and type in the pipe symbol (|) as the delimiter. The last screen will allow you to format the individual columns.
